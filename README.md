Full sound support (triangle, pulse 1, pulse 2, noise, DMC).

Mappers supported:

- MMC1 (1)
- UxROM (2)
- CNRom (3)
- MMC3 (4)
- AxRom (7)
- MMC2 (9)
- Mapper 19 (19)
- GxRom (66)
